# Installation

This guide will walk you through the process of installing the music software on your computer, ensuring that everything is ready for a smooth first experience.

---

## System Requirements

Before installing, make sure your device meets the following minimum requirements:

- **Operating System**: Windows 10 (64-bit) or later, macOS 11 (Big Sur) or later
- **Processor**: Intel Core i3 or equivalent (ARM and Apple Silicon supported)
- **Memory**: 4GB RAM minimum (8GB recommended for advanced use)
- **Storage**: 500MB available disk space (SSD recommended for better performance)
- **Audio**: Standard audio output device (internal or external)

## Installation Steps

### 1. Download the Installer

Head to our official [Download Page](/download) and choose the correct version for your system:

- `.exe` for Windows
- `.dmg` for macOS

### 2. Run the Installer

- **Windows**: Double-click the `.exe` file and follow the installation wizard.
- **macOS**: Open the `.dmg` file, drag the app icon into the `Applications` folder.

You may need to allow the installation through your system's security settings.

### 3. Launch the Software

After installation:

- On **Windows**: Open the app from the Start Menu or Desktop shortcut.
- On **macOS**: Find it in Launchpad or `/Applications`.
