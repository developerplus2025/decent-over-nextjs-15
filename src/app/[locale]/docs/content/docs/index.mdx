# Installation

Follow these steps to install the music software on your device.

## System Requirements

- **Operating System**: Windows 10 or later, macOS 11 or later
- **Processor**: Intel or ARM, 64-bit
- **Memory**: Minimum 4GB RAM
- **Storage**: At least 500MB of free space

## Installation Steps

1. Download the installer from the [Download Page](/download).
2. Run the installer and follow the setup instructions.
3. After installation, launch the software from the desktop or start menu.

## Post-Installation Checklist

- Make sure the software launches successfully.
- Test audio output to confirm playback works.
- Sign in if prompted.
